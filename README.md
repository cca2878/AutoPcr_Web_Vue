# AutoPcr Web Vue

New vue3-based frontend for @cc004/autopcr.


## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Compile and Minify for Production

```sh
npm run build
```
